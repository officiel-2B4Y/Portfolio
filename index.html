About me

My goals

Mes futures compétences créatives et mes expériences à acquérir.

Vous trouverez ci-dessous les langages de programmation qui m'aideront à parfaire mes futures connaissances avec le pourcentage selon lequel cela m'aidera dans mes projets.

1. Python (60%): Python est un langage polyvalent, facile à apprendre et largement utilisé dans divers domaines. Il est idéal pour le développement web, l’analyse de données, l’apprentissage automatique (machine learning) et l’automatisation de tâches.

2. XAML (50%): XAML est un langage de balisage utilisé pour la création d’interfaces utilisateur dans les applications de plateforme Microsoft, notamment dans le développement d’applications de bureau et d’applications mobiles.

3. Java (30%): Java est un langage de programmation couramment utilisé pour le développement d’applications d’entreprise, de logiciels pour appareils mobiles (Android) et de jeux vidéo.

4. C++ (70%): C++ est un langage puissant et performant, souvent utilisé pour le développement de logiciels système, de jeux, de moteurs graphiques et de logiciels embarqués.

5. MySQL (90%): MySQL est un système de gestion de bases de données relationnelles très populaire. Il est utilisé pour stocker, gérer et interroger des données dans de nombreux types d’applications.
